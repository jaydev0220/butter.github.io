# 成為學員

歡迎加入笨錢社群。請依照以下步驟完成註冊與驗證。完成後即可開始使用社群、課程與相關學習資源。

---

## 加入流程

### Step 1｜註冊新的 Bybit 帳戶

請使用下方指定邀請連結註冊新的 Bybit 帳戶

- 註冊連結：`<affiliate_link>`
- 邀請碼：`8TER`

#### 注意事項

- 請確認註冊時綁定邀請碼: `8TER`
- 建議使用尚未註冊過 Bybit 的 Email 或手機號碼
- 若您已經有 Bybit 帳戶，請先確認是否需要進行身分轉移

### Step 2｜完成 KYC 與入金

註冊完成後，請在 Bybit 帳戶內完成以下事項：

1. KYC 身分驗證
2. 帳戶安全設定
3. 入金流程

### Step 3｜查詢您的 Bybit UID

完成註冊、KYC 與入金後，請登入 Bybit，查詢您的 UID

#### UID 查詢方式

1. 登入 Bybit 帳戶
2. 進入個人帳戶頁面
3. 找到 UID / User ID
4. 複製完整 UID

請確認 UID 正確，避免影響資格審核。

### Step 4｜加入 Discord 並留言 UID

加入 Discord 後，到指定 UID 專區完成留言驗證

- Discord 社群連結：`<discord_link>`

## 常見問題

### 1. 我已經有 Bybit 帳戶，還可以加入嗎？

可以，但請先確認原帳戶是否已綁定其他邀請人。若需要轉移身分，請參考身分轉移教學影片：

- 身分轉移教學影片：`<transfer_video_link>`

### 2. 我填錯或留言錯 UID 怎麼辦？

請在 Discord UID 專區重新留言並註明。

### 3. 沒有完成 KYC 或入金可以加入嗎？

課程影片皆為免費且公開的。若您想與其他學員交流討論，您必須完成 KYC 與入金才能加入社群。
